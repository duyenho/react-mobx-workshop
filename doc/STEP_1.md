# Step 1: Add a new todo

## Requirements

1. User can input some words
2. When user presses "Enter", should put the new input into todo list
3. Do not accept blank content


## You should know

1. Create a store `Todos`
2. `@observable`
3. `<Provider/>`,
4. `@inject` and `@observer`


## Stores

```
┌─────────────────────────┐
│          Todos          │
├─────────────────────────┤
│ todos: Array            │
├─────────────────────────┤
│ addTodo()               │
└─────────────────────────┘
```
